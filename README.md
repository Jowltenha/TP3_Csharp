# Exercícios de Programação Orientada a Objetos em C#
- Este repositório contém uma série de exercícios práticos desenvolvidos para consolidar os fundamentos da Programação Orientada a Objetos (POO) utilizando a linguagem C#. O conteúdo abrange desde conceitos básicos até a manipulação de estados, construtores e cálculos matemáticos.

## 📌 Conteúdo do Repositório

- O projeto está dividido em três grandes contextos práticos:

1. Venda de Ingressos: Foco em atributos, métodos de atualização, Getters/Setters e Construtores.
2. Sistema Acadêmico: Foco em modelagem de processos (matrícula) e transição de estados.
3. Geometria Computacional: Foco em lógica matemática e retorno de valores.

## 📂 Detalhamento dos Exercícios
1. Fundamentos de POO (Exercício 1)

Explicação teórica e prática dos pilares da POO:
- Classe: O molde/planta (ex: Livro).
- Objeto: A instância real do molde.
- Campos/Atributos: Características (ex: Titulo, Autor).
- Métodos: Ações (ex: ExibirProgresso()).

2. Sistema de Venda de Ingressos (Exercícios 2 ao 6)
   
Modelagem de uma classe Ingresso para gerenciar eventos musicais.
- Atributos: nomeDoShow, preco, quantidadeDisponivel.
- Encapsulamento Manual: Implementação de métodos Get e Set para proteção de dados.
- Construtores: Implementação de inicialização obrigatória, garantindo que nenhum ingresso seja criado sem os dados essenciais.

3. Matrícula de Faculdade (Exercícios 7 ao 9)
   
Modelagem de fluxo acadêmico através da classe Matricula.
- Gerenciamento de Estado: Métodos Trancar() e Reativar() que alteram a propriedade Situacao.
- Relatórios: Método para exibir o status consolidado do aluno no console.

4. Formas Geométricas (Exercícios 10 ao 12)
   
Cálculos de áreas e volumes sem o uso de herança, focando em métodos funcionais.
- Círculo: Cálculo de área ($A = \pi \cdot r^2$).
- Esfera: Cálculo de volume ($V = \frac{4}{3} \cdot \pi \cdot r^3$).
- Uso da Classe Math: Aplicação de Math.PI para precisão nos resultados.

## 🛠️ Tecnologias Utilizadas
1. Linguagem: C#
2. Plataforma: .NET
3. Conceitos Aplicados: Classes, Objetos, Atributos, Métodos, Construtores, Getters/Setters, Lógica Matemática.
 
### Aprendizado Adquirido
Ao concluir estes exercícios, foi possível compreender como a orientação a objetos facilita a organização de sistemas complexos, permitindo que cada "parte" do software (como um Ingresso ou uma Matrícula) seja independente, segura e fácil de manter.
